# 💪 Strength Training

A 12-week undulating periodized strength training logbook built in Microsoft Excel and compatible with Google Sheets.

This project was created to simplify strength training by automatically calculating training weights from estimated 1RM values. Instead of calculating percentages manually before every workout, the workbook generates target weights automatically.

The goal is simple:

> Open the workbook → Check the target weight → Train → Log your session.

# Why I Built This

I created this workbook because I wanted a simple system that removes manual calculations during training.

Instead of spending time calculating percentages before every workout, I only update my estimated 1RM on the dashboard and let the workbook calculate the target weights automatically.

This project also helped me improve my Excel skills by building formulas, automation, and structured training templates.
